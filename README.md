# Student-Placement-Management-System
A web-based Student Placement Management System developed using FastAPI, SQLAlchemy, SQLite, Jinja2 Templates, and Bootstrap 5.  The system helps training institutes, placement cells, and colleges manage student placement activities efficiently through an interactive dashboard and management portal.
Features
Student Registration & Management
Add, Update, Delete Student Records
Placement Tracking
Internship Tracking
PPO (Pre-Placement Offer) Tracking
Company & Package Management
Placement Analytics Dashboard
Average CGPA Calculation
Highest Package Calculation
Placement Percentage Tracking
Student Search Functionality
Responsive Bootstrap UI
